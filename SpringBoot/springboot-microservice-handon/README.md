# Springboot Microservice With Vue


## VUE Environment Configuration

```
//1. 设置淘宝镜像
npm config set registry http://registry.npm.taobao.org/

//2. 全局安装nuxt [参考](https://nuxtjs.org/guide/installation)
npm i nuxt -g
npm i nuxt --save

//3. 使用npx创建项目
npx create-nuxt-app <project-name>


```
